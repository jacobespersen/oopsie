# Code of Conduct

## Expected Behavior

Everyone participating in this project — whether filing issues, submitting pull requests, or commenting — is expected to:

- Be respectful and considerate
- Accept constructive criticism gracefully
- Focus on what is best for the project
- Show empathy towards others

## Unacceptable Behavior

- Harassment, trolling, or personal attacks
- Sexualized language or imagery
- Publishing others' private information without permission
- Other conduct that would be inappropriate in a professional setting

## Enforcement

Unacceptable behavior can be reported to **jacobespersen@gmail.com**. Reports will be reviewed and addressed appropriately.
